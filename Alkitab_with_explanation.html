Parser Alkitab Sederhana
Experimen untuk memparsing Ayat-ayat Alkitab sederhana
Parser Ayat Akitab ini masih sangat sederhana dan hanya untuk format "kitab pasal:ayat" dan "kitab pasal:ayat1-ajat2"

contohnya "Mat 3:5" atau "Mat 3:4-8" untuk Mobile karena bagi saya masih sulit mencari padanan mouseover/mouseout, tetap menggunakan tap yaitu tap pada sasaran untuk display dan tap diluar sasaran untuk clear

Variasi memungkinkan yang bisa dilakukan : - lewat kotak input. - lewat parameter Alkitab.html?Yoh 3:16, Alkitab.html?Yoh. 3:16, Alkitab.html?Joh 3:16. - Lewat mouse click via anchor href. - Lewat mouseover via addEventListener. - Lewat mouseover via program otomatis

ini lewat program otomatis [Mat. 3:12-24 - Tit. 1:2]  ini lewat addEventListener [Kej. 1:2]  dan ini lewat mouseclick [ Yoh. 3:16 - Mat. 2:4]
[Enter] to display Ayat; Click disini untuk Hapus Display atau tekan [Esc]
